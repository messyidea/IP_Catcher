IP_Catcher
===

function
----------
    抓取本机ip并显示相关信息。


about codes
------------
    工程为VS2010工程
    工程名叫IP_Catcher。
    Socket初始化在CIP_CatcherDlg类的OnBnClickedOk函数中。
    显示ip数据在CIP_CatcherDlg类的show_Ip函数中。
    Ip格式在IP_Info.h头文件中。
